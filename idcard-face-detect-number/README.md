通过身份证上的头像,定位身份证上的信息位置,并返回身份证号.</br>
思路:</br>
通过face-detect得到身份证上的面部识别框face-rect;</br>
通过face-rect的宽高作为比例，计算出身份证号的大体相对位置和大小;</br>
通过对身份证号区域的ocr，得到其中的号码;</br>
注:</br>
因为照片中的面部有相对较大或者较小之分,故加入身份证号正则判断，并进行:</br>
一般/较小/较大面部比例 重获身份证号区域。</br>
由于图片本身质量和在二值化和去噪过程中的信息损失，识别结果存在可理解的误差,故识别出的字母进行人工的数字转换,例如:</br>
(" ","")</br>
("O","0")</br>
("U","0")</br>
("D","0")</br>
("Z","2")</br>
("S","5")</br>
("s","5")</br>
("o","6")</br>
("f","7")</br>
("H","11")</br>

运行命令:</br>
python idbyface.py ids.png </br>
